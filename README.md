# Campus_placement_insights

A five-year campus placement analysis project featuring:

- **Raw PDF placement reports**
- **Extracted and cleaned CSV datasets**
- **PostgreSQL data modeling** (schema, joins, transformations)
- **Power BI dashboard** for company-wise, branch-wise, and year-wise insights

This repository serves as a complete **data engineering + BI pipeline** demonstration.

## Data folders

- `data_raw_pdf/` – Raw placement reports in PDF format (2019–2023), directly downloaded from the college website.  
- `data_clean_csv/` – Cleaned CSV files extracted from the PDF tables (companies, offers, CTC, and student placement details).
